# 🌱 OptiCrop – Demo Overview

OptiCrop is a smart agriculture web application that helps users find the best crop to grow based on environmental conditions.

---

## 🎯 What This Demo Shows

* Entering environmental data (temperature, humidity, rainfall)
* Clicking **"Recommend Crop"**
* Instant crop prediction using machine learning
* Automatic scrolling to results section
* Display of:

  * 🌾 Recommended crop
  * 🖼️ Crop image
  * 📋 Helpful crop details

---

## ⚡ Key Highlights

* Simple and user-friendly interface
* Fast and accurate crop recommendations
* Visual output with crop images
* Smooth user experience with auto-scroll

---

## 🧪 Example Flow

1. User enters input values
2. Clicks **Recommend Crop**
3. System predicts crop instantly
4. Page scrolls to show result
5. Crop image appears beside recommendation
